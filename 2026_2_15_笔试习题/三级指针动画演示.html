const char* c[] = { "ENTER","NEW","POINT","FIRST" };
const char** cp[] = { c + 3,c + 2,c + 1,c };
const char*** cpp = cp;
printf("%s\n", **++cpp);
printf("%s\n", *-- * ++cpp + 3);
printf("%s\n", *cpp[-2] + 3);
printf("%s\n", cpp[-1][-1] + 1);
字符串层（最终指向）
ENTER
NEW
POINT
FIRST
c数组（const char*）
c[0] → ENTER
c[1] → NEW
c[2] → POINT
c[3] → FIRST
cp数组（const char**）
cp[0] → c[3]
cp[1] → c[2]
cp[2] → c[1]
cp[3] → c[0]
cpp指针（const char***）
cpp → cp[0] ↓
